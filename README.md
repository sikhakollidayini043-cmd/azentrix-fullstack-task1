# Personal Budget Tracker
Hello, this is my Personal Budget Tracker Web Application.

The purpose of this project is to help users manage their income and expenses in one place. Users can add, edit, and delete transactions, view their financial summary, and track spending through charts. The application stores data using localStorage, so information remains available even after refreshing the page.
## Problem Statement
Small businesses and individuals struggle to track daily income and expenses.

## Features
- Add income and expenses
- Edit transactions
- Delete transactions
- Monthly summary dashboard
- Pie chart visualization
- Data stored in localStorage

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run
1. Clone repository
2. Open index.html in browser

# Personal Budget Tracker

## Problem Statement
Small businesses and individuals struggle to track daily income and expenses.

## Features
- Add income and expenses
- Edit transactions
- Delete transactions
- Monthly summary dashboard
- Pie chart visualization
- Data stored in localStorage

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

## How to Run
1. Clone repository
2. Open index.html in browser
